Tasky es una aplicación diseñada para ayudar a los desarrolladores en la gestión y seguimiento de sus proyectos. Esta plataforma ofrece herramientas potentes para crear, organizar, editar y eliminar tareas dentro de cada proyecto.

La idea de esta aplicación surgió porque quería organizarme con los proyectos que estaba haciendo en el curso CS50 de Harvard y no encontré ninguna aplicación que sea específica para programadores, por lo que quise hacer una yo para mi proyecto final.

Utiliza tecnologías modernas como Flask, SQLAlchemy y Jinja2 en el backend, proporcionando una estructura robusta y segura para la manipulación de datos. En el frontend, se emplean HTML, CSS y JavaScript para ofrecer una experiencia de usuario interactiva y fluida.

Con Tasky, los desarrolladores pueden crear proyectos individuales, añadir y eliminar tareas, así como marcarlas como completadas o pendientes. La aplicación está diseñada para ser intuitiva y fácil de usar, permitiendo a los usuarios centrarse en sus tareas sin complicaciones en la gestión del proyecto.

Es una herramienta esencial para cualquier desarrollador que busque optimizar su productividad, ya que ofrece una manera sencilla de mantenerse organizado y enfocado en sus proyectos.
